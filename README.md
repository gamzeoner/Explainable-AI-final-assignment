# Explainable-AI-final-assignment

This project explores explainability methods for a heart disease prediction model.

Methods used:
- SHAP
- LIME
- Permutation Feature Importance
- Counterfactual analysis

Files:
- xai_final_assignment.ipynb
- heart.csv

How to run:
1. Open the notebook in Google Colab
2. Upload heart.csv
3. Run all cells

Libraries used:
- scikit-learn
- shap
- lime
- pandas
- matplotlib
